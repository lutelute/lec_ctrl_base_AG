# 第9章：LTIシステムの安定性 (詳細版)

本章では、制御工学において最も基本的かつ重要な概念である「安定性」を定義し、特性方程式の係数から安定性を判別する方法（ラウス・フルビッツの安定判別法）を体系的に学びます。

---

## 9.1 安定性の定義

### 9.1.1 BIBO安定と漸近安定
1.  **BIBO安定 (Bounded-Input Bounded-Output Stability)**
    - 定義：有界な入力（無限大にならない入力）に対して、出力がつねに有界であること。
    - 条件：インパルス応答 $g(t)$ が絶対可積分であること。 $\int_{0}^{\infty} |g(t)| dt < \infty$

2.  **漸近安定 (Asymptotic Stability)**
    - 定義：初期値応答が時間とともにゼロに収束すること。
    - 線形時不変 (LTI) システムでは、BIBO安定と漸近安定はほぼ等価（可制御・可観測であれば完全に等価）です。

### 9.1.2 極の位置と安定性
伝達関数 $G(s)$ の極（特性方程式 $D(s)=0$ の根）を $p_i$ とします。
インパルス応答は $g(t) = \sum A_i e^{p_i t}$ の形になります。

 $p_i = \sigma_i + j\omega_i$ とすると、 $|e^{p_i t}| = |e^{\sigma_i t} e^{j\omega_i t}| = e^{\sigma_i t}$ なので、振る舞いは実部 $\sigma_i$ で決まります。

- **安定 (Stable)**: 全ての極の実部が負 ($\sigma_i < 0$)。 $\lim_{t\to\infty} g(t) = 0$。
- **安定限界 (Marginally Stable)**: 実部がゼロの極（虚軸上の極）があり、かつ重根でない。他はすべて負。持続振動または定数。
- **不安定 (Unstable)**: 実部が正の極 ($\sigma_i > 0$) が少なくとも1つある。または虚軸上の極が重根。

## 9.2 ラウス・フルビッツの安定判別法 (Routh-Hurwitz Criterion)

高次方程式の根を直接計算せずに、係数の符号だけで安定性を判別する方法です。
特性方程式を以下とします。
$$ a_n s^n + a_{n-1} s^{n-1} + \dots + a_1 s + a_0 = 0 \quad (a_n > 0) $$

### 9.2.1 必要条件
安定であるための **必要条件** は、
> すべての係数 $a_i$ が存在し、かつ同符号（すべて正）であること。

これは、「根と係数の関係」から導かれます。
$(s+p_1)(s+p_2)\dots = 0$ を展開したとき、すべての $p_i$ の実部が正（極の実部は負）なら、係数はすべて正の和になるからです。
もし係数に0や負のものがあれば、その時点で「不安定」または「安定限界」です。

### 9.2.2 ラウスの数列（十分条件の確認）
必要条件を満たしていても、不安定な場合があります（例：振動しながら発散）。
以下の **ラウス表 (Routh Array)** を作成します。

| 行        |           |           |               |
| :-------- | :-------- | :-------- | :------------ |
| $s^n$     | $a_n$     | $a_{n-2}$ | $a_{n-4}$ ... |
| $s^{n-1}$ | $a_{n-1}$ | $a_{n-3}$ | $a_{n-5}$ ... |
| $s^{n-2}$ | $b_1$     | $b_2$     | $b_3$ ...     |
| $s^{n-3}$ | $c_1$     | $c_2$     | ...           |
| ...       | ...       |           |               |

新しい係数 $b_i, c_i$ の求め方：
$$ b_1 = \frac{a_{n-1} a_{n-2} - a_n a_{n-3}}{a_{n-1}} $$
$$ b_2 = \frac{a_{n-1} a_{n-4} - a_n a_{n-5}}{a_{n-1}} $$
$$ c_1 = \frac{b_1 a_{n-3} - a_{n-1} b_2}{b_1} $$

視覚的には、「直上の2行のクロス積」を「直上の左端」で割ったものの符号反転、と覚えます。

> **ラウスの安定判別法**
> ラウス表の **第1列** ($a_n, a_{n-1}, b_1, c_1, \dots$) の符号がすべて正であれば、システムは安定である。
> 符号の変化回数は、不安定極（右半平面の極）の個数に等しい。

### 9.2.3 具体例

**例題 1**: $s^3 + 2s^2 + 3s + 4 = 0$
1.  必要条件: 係数は 1, 2, 3, 4。すべて正。OK。
2.  ラウス表:
    - $s^3$: 1, 3
    - $s^2$: 2, 4
    - $s^1$: $\frac{2\times 3 - 1\times 4}{2} = \frac{6-4}{2} = 1$
    - $s^0$: $\frac{1\times 4 - 2\times 0}{1} = 4$
    第1列は $1, 2, 1, 4$。すべて正。
    $\therefore$ **安定**。

**例題 2**: $s^3 + s^2 + 2s + 8 = 0$
1.  必要条件: 1, 1, 2, 8。OK。
2.  ラウス表:
    - $s^3$: 1, 2
    - $s^2$: 1, 8
    - $s^1$: $\frac{1\times 2 - 1\times 8}{1} = -6$
    - $s^0$: $\frac{-6\times 8 - 1\times 0}{-6} = 8$
    第1列は $1, 1, -6, 8$。符号変化は $1 \to -6$ (1回)、$-6 \to 8$ (1回) で計2回。
    $\therefore$ **不安定**（右半平面に極が2個ある）。

实际に解くと $(s+2)(s^2-s+4)=0$、極は $-2, 0.5 \pm j1.93$。確かに実部正の極が2つあります。

## 9.3 演習問題

**問題**:
特性方程式 $s^4 + 2s^3 + 3s^2 + 4s + K = 0$ が安定となるゲイン $K$ の範囲を求めよ（$K>0$）。

**解答**:
ラウス表を作成します。
- $s^4$: 1, 3, $K$
- $s^3$: 2, 4
- $s^2$: $\frac{2\times 3 - 1\times 4}{2} = \frac{2}{2} = 1$, $\quad \frac{2\times K - 1\times 0}{2} = K$
  （$s^2$行: $1, K$）
- $s^1$: $\frac{1\times 4 - 2\times K}{1} = 4 - 2K$
- $s^0$: $\frac{(4-2K)K - 1\times 0}{4-2K} = K$

第1列がすべて正である条件：
1.  $1 > 0$ (OK)
2.  $2 > 0$ (OK)
3.  $1 > 0$ (OK)
4.  $4 - 2K > 0 \implies 2K < 4 \implies K < 2$
5.  $K > 0$

したがって、安定範囲は **$0 < K < 2$** です。
($K=2$ のとき安定限界で持続振動します)。
